# TinyERP v2 SDK

SDK para integração com API do TinyERP v2

## Instalação

```bash
yarn install
```

## Desenvolvimento

```bash
# Executar em modo dev
yarn dev

# Build
yarn build

# Testes
yarn test
yarn test:watch
yarn test:coverage
```

## Estrutura

```
src/
├── index.ts          # Entry point
└── index.test.ts     # Tests
```
